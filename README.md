# 🐦 Bird Species Observation Analysis — Streamlit Dashboard

**Domain:** Environmental Studies · Biodiversity Conservation · Ecology  
**Skills:** Data Cleaning · EDA · SQL · Plotly · Streamlit

---

## 📋 Project Overview

This interactive Streamlit dashboard analyzes the **distribution and diversity of bird species** across **Forest** and **Grassland** habitats, covering all 11 National Park administrative units:

`ANTI · CATO · CHOH · GWMP · HAFE · MANA · MONO · NACE · PRWI · ROCR · WOTR`

---

## 🚀 Quick Start (Local Setup)

### Step 1 — Prerequisites
Make sure you have **Python 3.9+** installed.

### Step 2 — Install dependencies
Open a terminal in this project folder and run:

```bash
pip install -r requirements.txt
```

### Step 3 — Run the app
```bash
streamlit run app.py
```

Your browser will open automatically at **http://localhost:8501**

---

## 📂 Dataset

Download the dataset from the project Google Drive link:

> **Bird_Monitoring_Data_FOREST.XLSX**  
> **Bird_Monitoring_Data_GRASSLAND.XLSX**

Place both files anywhere on your computer.  
When the dashboard opens, use the **sidebar** to upload them.
